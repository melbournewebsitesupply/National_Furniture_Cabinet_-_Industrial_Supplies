Get Started

How to Place an Order

  • Take note of the item's product code

    The product description code can usually be found in the first left hand column of the item's listing.

  • Click on the corresponding button that matches the product description code.

    This will take you to the checkout, where you can adjust quantity and add extra order details for the selected item.

    Please Note: The delivery fee is not included at this stage.

  • Get it delivered

    A $20 flat-delivery-fee can be paid through the payment link listed on the purchase receipt sent to your email (or click here). This will complete your order and enable it to be processed.

Please Note: Only one delivery fee payment of $20 is required, regardless of the quantity of items / different types purchased.
